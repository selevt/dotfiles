Dotfiles
========

My dotfiles and scripts

Usage
-----
from inside the dotfiles directory::

    ./link.sh

this will create symlinks to home directory for every file inside the config
directory

Content
-------
* VIM config and plugins:
 * color: molokai
 * nerd commenter
 * nerd tree
 * vim-latex
 * vst (vim restructured text)
